Cushitic Database

The (East) Cushitic Database is a comprehensive resource for researchers, linguists, and anyone interested in Cushitic languages. It contains detailed information on phonology, morphology, and lexicons of various Cushitic languages.

Features

  • Comprehensive word lists
  • Phonological inventories
  • Morphological paradigms
  • Comparative Data
  • Historical linguistic data

Languages Included (Mainly EC)

  • Saho, Afar
  • Rendille, Somali, Maay, Boni
  • Arbore, Dhaasanac, Elmolo, Bayso
  • Hadiyya, Kambaata, Sidaama, Gedeo, Burji
  • Oromo, Diraytata, Mositacha, Konso
  • Gawwada, Ts'amakko, Yaaku
  • Dahalo

Database Access

The Cushitic Database is currently under development and will be available for research purposes in the near future. If you're interested in accessing the database or contributing to its development, please contact us using the form below.
